# Data — Cleaned Datasets

This folder contains the cleaned, ready-to-use datasets for each indicator, exported from the cleaning process in `../excel/`.

## Files
- `clean_debt_rate.csv`
- `clean_electricity_rate.csv`
- `clean_inflation_rate.csv`
- `clean_naira_value.csv`
- `clean_poverty_rate.csv`
- `clean_unemployment_rate.csv`
- `clean_violence_rate.csv`

Each file covers Nigeria's yearly values for that indicator from 2012 to 2025.

## Notes
- These are the files imported directly into the R scripts in `../r/`
- Raw sources (World Bank, DMO Nigeria, ACLED, Investing.com) are linked in the main [README](../README.md)
